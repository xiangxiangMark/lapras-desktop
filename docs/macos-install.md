# Lapras macOS 邀请测试版安装说明

适用设备：搭载 Apple Silicon 的 Mac，也就是 M1、M2、M3、M4 系列芯片。

系统要求：macOS 13 或更高版本。

当前版本：Lapras 1.0.0 邀请测试版。此版本暂未进行 Apple 官方公证，因此首次打开时 macOS 可能会提示无法验证开发者。这是当前测试版的预期现象。

## 安装步骤

1. 下载邀请链接中的 `Lapras-1.0.0-arm64.dmg`。
2. 双击打开 DMG 文件。
3. 将 `Lapras` 拖到 `Applications` 文件夹。
4. 打开 `Applications` 文件夹，找到 `Lapras`。
5. 不要直接双击。请按住 `Control` 键点击 `Lapras`，或者右键点击 `Lapras`。
6. 在菜单中选择 `打开`。
7. 如果系统弹出安全提示，再次点击 `打开`。

完成这一步后，以后就可以像普通应用一样直接打开 Lapras。

## 如果系统仍然阻止打开

1. 先尝试打开一次 `Lapras`，看到拦截提示后点击 `好`。
2. 打开 macOS `系统设置`。
3. 进入 `隐私与安全性`。
4. 向下找到关于 `Lapras` 被阻止打开的提示。
5. 点击 `仍要打开`。
6. 输入 Mac 登录密码或使用 Touch ID 确认。

## 首次启动配置

首次打开 Lapras 后，会进入初次引导流程：

1. 填入 DeepSeek API Key。
2. 选择是否登录网易云音乐。
3. 选择默认播放模式。

网易云登录可以跳过。跳过后，仍然可以在设置页中重新连接网易云账号。

## 卸载方式

1. 退出 Lapras。
2. 打开 `Applications` 文件夹。
3. 将 `Lapras` 移到废纸篓。

如果你还想删除本地数据，可以同时删除：

```text
~/Library/Application Support/Lapras
```

删除本地数据会清除 API Key、网易云登录状态、聊天记录、播放偏好和本地画像。

## 常见问题

### 为什么会提示无法验证开发者？

因为当前是邀请测试版，暂未进行 Apple 官方公证。macOS 默认会拦截未公证应用，所以需要通过右键打开或系统设置手动放行。

### 这个提示是否代表应用有问题？

不一定。它只说明应用没有经过 Apple 公证流程。请确认 DMG 来自 Lapras 官方邀请链接，不要安装来源不明的文件。

### 支持 Intel Mac 吗？

当前邀请测试版暂时只支持 Apple Silicon Mac，不支持 Intel Mac。

### 可以不登录网易云吗？

可以。Lapras 仍然可以聊天和使用基础推荐能力，但连接网易云后，推荐会更贴近你的歌单和最近播放。
