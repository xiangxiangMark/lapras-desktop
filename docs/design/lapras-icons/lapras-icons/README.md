## Design Language

- Shape: rounded cloud / water ripple mascot silhouette.
- Mood: calm, lightweight, companion-like, modern, not childish.
- Palette: cyan, aqua, blue, with a small violet accent.
- Motifs: night listening, water ripple, soft smile, tiny star highlight.
- Small-size rule: tray icon should use the simple one-color outline, not the gradient mascot.

## Production Notes

- Convert app icon from the main gradient mascot into PNG sizes: 16, 20, 24, 32, 48, 64, 128, 256, 512.
- Build Windows `.ico` from the cleaned 16-256 PNG set.
- Redraw the tray icon as a native SVG/ICO with a single stroke color for crisp 16px and 20px rendering.
- Use transparent PNG exports for in-app avatar and onboarding avatar.
- Keep player control icons as thin navy strokes with consistent optical weight.
- Prefer the revised chat avatar with the short upward smile; the first wave-mouth version can read as worried at small sizes.
